Grid Panel configured with a GeoExt.data.FeatureStore

This example shows an Ext grid loaded with features read from a GeoJSON document (../data/c14_region.json).

Because the layer and the store are bound to each other, the features loaded into the store are automatically added to the layer.
A GeoExt feature selection model is also used so that selecting rows in the grid selects features in the layer, and vice-versa.

See feature-grid.js.
